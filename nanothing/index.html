    NTSS LoRaWAN 1.0

    Release date: Coming Soon

    • 1

      Edge Autonomy

      NTSS LoRaWAN 1.0 is a supply chain traceability standard built on top of the LoRaWAN specification. It provides the interface between sensor data and the cloud, allowing any LoRa-enabled data logger to autonomously operate between two locations. The standard features device point to point wireless management, ensuring that all the logged data is securely and reliably provided to any application through our API.

    • 2

      Ease of Integration

      The standard for the edge is built on top of Semtech's LoRaWAN end-device stack reference implementation (LoRaMAC-node) and can be easily integrated into any LoRa-enabled device. It has built-in support for a variety of microcontrollers and a user friendly interface for adding more. On the cloud side, everything is handled by us and the sensor data and device information are available through our APIs.
